〇使用方法
　①「url.txt」を開き、開きたいHPのURLを一行ずつ入力する。各々は「""」で括る。（「url_example.txt」のようにする。）
　②文字コード：SJISで保存する。（そうしないと日本語がURLへ正しく反映されない。）
　③「openChrome.bat」を実行すると、chromeが起動し、「url.txt」内のHPがタブで開かれる。
　④「openChrome.bat」のショートカットをスタートアップフォルダへ貼り付ければ便利。

〇作成意図
　・chromeの拡張機能「Snoozz - Snooze Tabs & Windows for later」の代替。
