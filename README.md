# 🚀 openChromeInMultiTabs

## 概要 (What's This?)
`openChromeInMultiTabs` は、Windows PCの起動時に、あらかじめ指定した複数のウェブページを、新しいGoogle Chromeウィンドウで自動的に開くためのシンプルなバッチファイルです。毎日の情報収集や定型業務の開始を、ワンクリックなしで自動化します。

## ✨ 主な特徴 (Features)
このツールの最大の価値は、**通常のChrome起動設定と併用できる点**にあります。

Windowsのスタートアップフォルダに、
1.  元からある「Google Chrome」のショートカット（前回終了時のタブを復元する設定など）
2.  このツールの「`openChrome.bat`」のショートカット

この2つを両方置くことで、PC起動時に**「① 普段使いのウィンドウ」**と**「② 特定用途のウィンドウ（例: プロジェクト関連ページ、情報収集サイトなど）」**という、2つの異なるウィンドウを同時に自動で立ち上げることが可能になります。

このユニークな機能により、あなたのPC起動後の作業をよりスムーズに、より効率的にします。

## 使い方 (Usage)
セットアップは非常に簡単です。以下の手順に従ってください。

1.  **リポジトリのダウンロード**
    - このページの上部にある緑色の「**<> Code**」ボタンをクリックし、「**Download ZIP**」を選択してリポジトリのファイルをダウンロードします。
    - ダウンロードしたZIPファイルを解凍し、任意の場所にフォルダを配置してください。

2.  **URLリストの編集**
    - `url.txt` をメモ帳などのテキストエディタで開きます。
    - 開きたいウェブサイトのURLを、1行に1つずつ入力してください。
    - 各URLは必ずダブルクォーテーション（`""`）で囲む必要があります。
    - 編集の参考に `url_example.txt` をご覧ください。

    **（例: url.txt）**
    ```text
    "https://github.com"
    "https://x.com"
    "https://www.google.com"
    ```

3.  **文字コードを指定して保存**
    - **【最重要】** `url.txt` を保存する際、文字コードを **SJIS (Shift_JIS)** に指定してください。
    - これを怠ると、URLに日本語が含まれる場合に文字化けし、ページを正しく開けません。
    - （メモ帳の場合: 「ファイル」 > 「名前を付けて保存」を選択し、保存ボタンの隣にあるエンコードのドロップダウンメニューから `ANSI` または `Shift_JIS` を選択します）

4.  **動作確認**
    - `openChrome.bat` をダブルクリックします。
    - `url.txt` に記述したURLが、新しいChromeウィンドウの各タブで開かれれば成功です。

5.  **スタートアップに登録（推奨）**
    - `openChrome.bat` のショートカットを作成します。
    - 作成したショートカットをWindowsのスタートアップフォルダに配置すると、PCを起動するたびに自動で実行され、非常に便利です。
    - **スタートアップフォルダの開き方:**
        - `Windowsキー + R` を押して「ファイル名を指定して実行」ウィンドウを開きます。
        - `shell:startup` と入力して `Enter` キーを押すと、現在のユーザー用のスタートアップフォルダが開きます。ここにショートカットを移動してください。

## ⚠️ 注意点 (Important Notes)
- `url.txt` を編集・保存する際は、必ず文字コードを **SJIS (Shift_JIS)** にしてください。UTF-8など他の文字コードでは正しく動作しません。
- Google ChromeがPCにインストールされている必要があります。

## 📂 ファイル構成 (File Structure)
```
openChromeInMultiTabs/
├── openChrome.bat      # メインの実行ファイル
├── url.txt             # 開きたいURLを記述するファイル（ユーザーが編集）
└── url_example.txt     # url.txtの記述例ファイル
```
